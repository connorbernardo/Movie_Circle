# Movie_Circle
Flask web application for rating and tracking movies

**Author:** Connor Bernardo

---

## Main Page
Start main page of web application
- `index.html` - main page

## Login Page
Web page to login or create new users connected to database
- `login.html` - login and registration page

## rate movie page
search movies to log
-`rate_movie.html` - search movies then add them to your profile

---

## How to Run

1. Navigate to application folder
    ```
    cd "application folder name"
    ```
2. Install dependencies
    ```
    pip install -r requirements.txt
    ```
3. Run App
    ```
    python app.py
    ```
